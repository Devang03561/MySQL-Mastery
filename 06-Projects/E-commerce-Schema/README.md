# E-Commerce Database Schema

A comprehensive database design for a typical e-commerce platform. Covers users, products, orders, and reviews.

## Tables
- `users` – Customers and admins.
- `categories` – Product categories.
- `products` – Items for sale.
- `orders` – Order header.
- `order_items` – Line items per order.
- `reviews` – Customer product reviews.

## How to Use
1. Execute `ecommerce_schema.sql` in your MySQL environment.
2. Use the schema to practice complex queries, multi-table joins, and reporting.

## 🚀 How to Use This Repository

1. **Install MySQL** – If you haven't already, follow the guide in `01-Basics/01-Installation-Guide.md`.
2. **Start from the beginning** – Work through each folder sequentially.
3. **Run the SQL files** – Open your MySQL client (CLI, Workbench, or DBeaver) and execute the files.
4. **Experiment** – Change the queries, add your own data, and test edge cases.
5. **Build the projects** – Head to `06-Projects/` and create the databases from scratch.
6. **Test yourself** – Solve the exercises in `07-Exercises/`.
